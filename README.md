# 制作アプリリスト

制作済み・制作中・今後作りたいアプリをブラウザだけで管理するWebアプリです。

## 使い方

`index.html` をブラウザで開いてください。入力内容はブラウザの LocalStorage に保存されます。

## 主な機能

- アプリ情報の登録・編集・削除
- 制作状況、ジャンル、優先度による絞り込み
- アプリ名、概要、備考のキーワード検索
- 更新日、作成日、優先度、制作状況、アプリ名での並び替え
- 状況別の件数表示
- CSVエクスポート
- 初回起動時のサンプルデータ
- スマートフォン、タブレット、PC対応

サーバーやログインは不要です。ブラウザごとにデータが保存されるため、別の端末やブラウザには自動で同期されません。
